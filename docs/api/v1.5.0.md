# Clockwork API v1.5.0

This is the versioned contract snapshot for API `1.5.0`.

## Scope Added Since 1.1

- Command/API UX 2.0:
  - `CommandSpec`-first command registration with typed args, middleware, auto-help/completion.
- Events 2.0:
  - `EventSubscriptionOptions` (`priority`, `ignoreCancelled`, `mainThreadOnly`)
  - async publish and tracing (`publishAsync`, `setTracingEnabled`, `eventTraceSnapshot`).
- Data/State API:
  - host mutation batches (`HostMutationBatch`, `HostMutationOp`, `HostMutationBatchResult`)
  - transactional rollback flow + migration-aware persistence schema.
- Assets/Content Pipeline:
  - asset bundle and validation contracts (`ResourcePackBundle`, `AssetValidationResult`)
  - animation/sound registry contracts.
- Plugin Network API:
  - `PluginNetwork` channels, schema versioning, backpressure, throughput quotas.
- Security/Isolation:
  - `host.mutation.batch` permission gate
  - adapter allow-list + capability permission scopes
  - plugin fault-budget telemetry in diagnostics/profile.
- In-game UI and chat ergonomics:
  - `ctx.notify(...)`, `ctx.actionBar(...)`, `ctx.closeUi(...)`, `ctx.broadcastNotice(...)`
  - improved menu/dialog rendering behavior in standalone runtime UI adapter.
