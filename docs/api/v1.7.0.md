# Clockwork API v1.7.0 (Freeze)

This file is the v1.7 contract freeze document for `clockwork-api` and `clockwork-host-api`.

## Freeze Rules

- Additive changes are allowed.
- Breaking removals/renames are blocked by default.
- Breaking exceptions require both:
  - explicit CI/Gradle release flag: `-PapiBreakingApproved=true`
  - explicit metadata in `docs/api/breaking-exceptions.properties`
