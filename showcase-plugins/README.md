# Showcase Plugins

This folder contains showcase plugin source projects linked from the website marketplace.

Marketplace page:
- `website/marketplace.html`

Plugins:
- Nick
- Godmode
- Veinminer
- Guarded
- SoftAnticheat
- Restocker

Notes:
- These are showcase/reference plugins for Clockwork API patterns.
- Some Minecraft-specific visuals (real skin swaps, tab-name rewrite) require host-level support beyond current standalone API.
- Each plugin folder contains `gigaplugin.yml`, Kotlin source, and a dedicated README.
