# Showcase: Scrible

Commands:
- `scribble-preview <text>`
- `scribble-save <name> <text>`
- `scribble-show <name>`
- `scribble-list`

Behavior:
- Supports legacy style codes like `&a`, `&l`, `&o`, `&n`.
- Adds modern gradient marker syntax `<g:from:to>text</g>` as preview formatting.
- Stores reusable snippets.
