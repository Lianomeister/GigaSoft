# Showcase: Enchanted

Commands:
- `enchanted-list [detail]`
- `enchanted-apply <preset> [player]`
- `enchanted-show [player]`
- `enchanted-clear [player]`
- `enchanted-duration <seconds>`
- `enchanted-create <name> <effect:amp,...>`
- `enchanted-delete <name>`

Behavior:
- Expanded PvP-focused preset library with persistent per-player assignments.
- Supports custom operator presets via command.
- Configurable effect duration and cleaner apply/clear flow.
